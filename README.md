## Getting Started

Install dependecies 

```bash
yarn install
```
```
create .env file with DATABASE_URL & JWT_SECRET
```

Seed db to remove users and add new ones
```bash
npx prisma db seed
```

Run the development server:

```bash
yarn dev
```
